# Энерджи — фитнес-клуб

Современный сайт фитнес-клуба на Next.js (App Router), TypeScript, Tailwind CSS, Framer Motion и xlsx.

## Запуск

1. `npm install`
2. `npm run dev`
3. Откройте `http://localhost:3000`

## Куда добавить изображения

Поместите фото клуба в папку:

- `public/images/`

Используемые пути в коде (можно расширять):

- `/images/gym-1.jpg`
- `/images/gym-2.jpg`
- `/images/gym-3.jpg`
- `/images/gym-4.jpg`

Если изображений пока нет, сайт показывает текстовые заглушки и не падает.

## Куда добавить Excel с расписанием

Поместите Excel-файл сюда:

- `data/schedule.xlsx`

Файл не добавлен в репозиторий специально — вы добавляете его вручную.

## Ожидаемый формат Excel

Рекомендуемые колонки (в первой строке):

- `день` / `день недели`
- `время`
- `программа`
- `тренер`
- `зал`
- `комментарий`
- `тип` (например: Силовые программы)

Если названия колонок отличаются, настройте маппинг в `lib/schedule.ts` (`columnMap`).

## Если Excel еще не добавлен

Сайт автоматически показывает демо-расписание и уведомление о том, что файл `data/schedule.xlsx` пока отсутствует.
