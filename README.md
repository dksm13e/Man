# Энерджи фитнес-клуб — Next.js проект

Современный адаптивный сайт фитнес-клуба на Next.js (App Router), TypeScript, Tailwind CSS, Framer Motion и xlsx.

## Запуск

1. `npm install`
2. `npm run dev`
3. Откройте `http://localhost:3000`

## Где менять временные изображения

- Массив галереи расположен в `lib/data.ts` (`galleryImages`).
- Сейчас используются внешние временные URL.

## Как заменить на реальные фото клуба

1. Добавьте изображения в `/public/images`, например:
   - `/public/images/gym-1.jpg`
   - `/public/images/gym-2.jpg`
2. В `lib/data.ts` замените URL на локальные пути:
   - `/images/gym-1.jpg`
   - `/images/gym-2.jpg`

## Excel-файл расписания

- Путь для файла: `/data/schedule.xlsx`
- Файл **не добавлен** в репозиторий специально.
- Чтение реализовано в `lib/schedule.ts` с помощью библиотеки `xlsx`.

### Ожидаемые колонки в Excel

- день недели
- время
- программа
- тренер
- зал
- комментарий

Можно использовать альтернативные названия колонок (например `day`, `time`, `program`) — маппинг настраивается в `columnMap` внутри `lib/schedule.ts`.

## Если Excel еще не добавлен

Сайт не падает: автоматически показывается демо-расписание с пометкой, что используется fallback.
