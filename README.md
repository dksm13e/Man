# Энерджи фитнес-клуб

Современный адаптивный сайт для бренда «Энерджи фитнес-клуб» на Next.js + TypeScript + Tailwind + Framer Motion.

## Запуск

```bash
npm install
npm run dev
```

Откройте: http://localhost:3000

## Где менять галерею

Временные URL изображений находятся в `app/page.tsx` в массиве `galleryImages`.

Чтобы перейти на реальные фото клуба, замените элементы массива на локальные пути:

- `/images/gym-1.jpg`
- `/images/gym-2.jpg`
- `/images/gym-3.jpg`
- и т.д.

## Где менять расписание

Данные расписания находятся в `data/schedule.ts`.

Текущая версия расписания вручную собрана по фото расписания и использует стабильную локальную структуру без Excel/xlsx.

Если фото расписания обновится:

1. Откройте `data/schedule.ts`.
2. Обновите соответствующие дни/время/названия/тренеров.
3. Сохраните файл и проверьте отображение в секции «Расписание».
