# Энерджи фитнес-клуб

Современный адаптивный сайт фитнес-клуба на Next.js (App Router) с темной брендовой подачей, интерактивным расписанием, галереей и формами заявок.

## Запуск проекта

1. Установите зависимости:
   ```bash
   npm install
   ```
2. Запустите dev-сервер:
   ```bash
   npm run dev
   ```
3. Откройте в браузере:
   `http://localhost:3000`

## Node.js

Проект рассчитан на Node.js `>=20`.

- Версия закреплена в `.nvmrc`
- Ограничение также указано в `package.json -> engines`

## Расписание из Excel

- Приложите Excel-файл в путь: `data/schedule.xlsx`
- Чтение и парсинг выполняются только на сервере в `lib/schedule.ts`
- Если файла нет или структура не подходит, автоматически используются fallback-данные

### Ожидаемые колонки Excel

- день недели
- время
- программа
- тренер
- зал
- комментарий

Поддерживается гибкий mapping заголовков, который можно быстро изменить в `columnMap`.
