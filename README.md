# Энерджи фитнес-клуб — сайт на Next.js

## Запуск проекта
1. `npm install`
2. `npm run dev`
3. Откройте `http://localhost:3000`

> Требуемая версия Node.js: 20+ (см. `.nvmrc`).

## Где менять временные изображения галереи
Сейчас галерея использует 10 внешних URL для проверки логики.

Файл:
- `lib/content.ts` → массив `galleryImages`

Чтобы заменить на реальные фото клуба, используйте локальные пути:
- `/images/gym-1.jpg`
- `/images/gym-2.jpg`
- ...

После этого добавьте файлы в папку `public/images`.

## Где находится Excel расписания
Путь к будущему файлу:
- `/data/schedule.xlsx`

Чтение Excel выполняется только на сервере в `lib/schedule.ts`.

## Ожидаемый формат Excel
В первой строке должны быть названия колонок:
- `день недели`
- `время`
- `программа`
- `тренер`
- `зал`
- `комментарий`

Допускаются альтернативные названия колонок (mapping уже добавлен в `lib/schedule.ts`).

## Что будет, если Excel пока нет
Сайт не падает: используется встроенный fallback (mock data) из `lib/content.ts`.
