# Энерджи фитнес-клуб — сайт на Next.js

## Запуск
1. `npm install`
2. `npm run dev`
3. Откройте `http://localhost:3000`

## Где менять контент
- **Реальные фото клуба**: замените файлы в `public/images/club/` (оставьте те же имена или обновите пути в `data/site-data.ts`).
- **Фото расписания**: замените файл `public/images/schedule/schedule-photo.svg` на новую фотографию расписания.
- **Структурированные карточки расписания**: обновите массив `scheduleCards` в `data/site-data.ts` под новую фотографию.
- **Номер телефона и tel-ссылку**: объект `phone` в `data/site-data.ts`.
- **Тексты сайта, тарифы, вопросы, контакты**: `data/site-data.ts`.
- **Список групповых программ**: `data/programs-data.ts`.

## Примечания по расписанию
- Раздел расписания построен **без Excel**.
- Источник — фото расписания + ручная структурированная выжимка в коде.
- Чтобы обновить расписание: замените фото и поправьте `scheduleCards` под новый снимок.
