# Энерджи фитнес-клуб — сайт на Next.js

## Запуск
1. `npm install`
2. `npm run dev`
3. Откройте `http://localhost:3000`

## Где менять данные
- Фото клуба (галерея и фоновые изображения): `public/images/club/`
- Фото расписания: `public/images/schedule/schedule-preview.svg` (замените на реальное фото расписания, лучше `.jpg` или `.png`)
- Телефон и ссылка звонка: `app/page.tsx` (поиск по `+7 (900) 000-00-00` и `tel:+79000000000`)
- Адрес клуба и ссылки маршрута: `app/page.tsx` (поиск по `Сарапул, Первомайская 34`)
- Данные расписания по дням: `data/siteData.ts` (`scheduleByDay`)
- Тексты сайта, FAQ, программы и тарифы: `app/page.tsx`

## Как обновить фото расписания
1. Добавьте новый файл в `public/images/schedule/` (например `schedule-2026.jpg`)
2. Обновите переменную `scheduleImage` в `data/siteData.ts`
3. При необходимости обновите массив `scheduleByDay` под новое фото расписания

## Примечание
Сейчас в проекте используются демонстрационные SVG-заглушки для фото (если реальные файлы не добавлены в репозиторий). Для продакшена замените их реальными фотографиями клуба и актуальным снимком расписания.
